void ripemd160 ( struct ripemd160 *  ripemd,

ripemd160 - return ripemd160 of an object. @ripemd160: the ripemd160 to fill in : pointer to memory, @size: the number of bytes pointed to by

The bytes pointed to by is RIPEMD160 hashed into @ripemd160. This is equivalent to ripemd160_init(), ripemd160_update() then ripemd160_done().

◆ ripemd160_init()

void ripemd160_init ( struct ripemd160_ctx *  ctx)

ripemd160_init - initialize an RIPEMD160 context. @ctx: the ripemd160_ctx to initialize

This must be called before ripemd160_update or ripemd160_done, or alternately you can assign RIPEMD160_INIT.

If it was already initialized, this forgets anything which was hashed before.

Example: static void hash_all(const char **arr, struct ripemd160 *hash) { size_t i; struct ripemd160_ctx ctx;

ripemd160_init(&ctx); for (i = 0; arr[i]; i++) ripemd160_update(&ctx, arr[i], strlen(arr[i])); ripemd160_done(&ctx, hash); }
